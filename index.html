InTheGame: Site para notícias de jogos

Este projeto é uma página web, criada utilizando as linguagens HTML E CSS. Seu propósito é auxiliar jogadores a encontrar notícias, dicas e lançamentos para seus jogos e franquias favoritas, trazendo diversos vídeos sobre diferentes tópicos.

A estrutura tem como base o HTML, que serve para definir a semântica e o layout do conteúdo, enquanto o CSS serve para trazer um estilo diferente e único ao site, com diferentes cores, fontes e disposição dos elementos.

PongJS: Diversão em JavaScript

Este jogo foi criado utilizando JavaScript, utilizando o clássico pong como inspiração, trazendo diversão e um sentimento de nostalgia para os jogadores.

O jogo utiliza de diversas variáveis e funções para garantir uma jogabilidade responsiva, divertida e justa. As funções verificam o local da bola, calculam os pontos e mapeiam o teclado para os dois jogadores.

AluraBooks: Literatura e tecnologia

Este projeto foi criado utilizando HTML, CSS e uma biblioteca JavaScript, buscando trazer uma experiência confortável, auxiliando a compra de livros para aprendizado tecnológico.

O site utiliza de diversos cards, abas e rolagem automática, através da biblioteca JSDelivr. Tudo é programado para garantir uma rolagem suave e um site responsivo para os seus usuários.
